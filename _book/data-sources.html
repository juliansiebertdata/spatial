3.4 Weather Data

  • WorldClim WorldClim is a set of global climate layers (climate grids) with a spatial resolution of about 1 km². It is based on weather station data and provides information on temperature and precipitation. The data is available as raster files and can be accessed via an API. There might be an R package to download data from WorldClimData.
    • Mean, minimum, and maximum temperature
    • Precipitation
    • Solar radiation
    • Wind speed
  • ERA5 ERA5 is a reanalysis dataset that provides global weather data at a resolution of 0.25°. It is based on a combination of weather station data, satellite data, and weather models. The data is available as raster files and can be accessed via an API. The API is documented here.
    • Temperature
    • Precipitation
    • Wind speed
    • Solar radiation
    • Humidity
  • CHIRPS CHIRPS is a global rainfall dataset that provides information on precipitation at a resolution of 0.05°. It is based on a combination of weather station data and satellite data. The data is available as raster files and can be accessed via an API. The API is documented here.

3.4.1 Precipitation Data

  • pRecipe Package in R makes a lot of common precipitation data accessible. There is a video on how to do this here

3.5 Soil Data

  • Soilgrids Soilgrids is a global gridded soil information system operating at a resolution of 250 m. It is based on machine learning and remote sensing and provides soil information for the entire globe. This data is not actually measured and was shown to underestimate extreme soil distributions. It is available as a raster file and can be accessed via an API. The API is documented here. And there is a soilgrids r package.
    • Percentage of sand, silt, clay
    • Bulk density
    • Organic carbon content
    • pH
    • Cation exchange capacity -WoSIS contains harmonized soil data from all over the globe. It is actually measured data. The way to access it with R is described here

3.6 Drought Data

  • GDO / CDI Global Drought Observatory with the Combined Drought Inidicator (CDI) provides information about changes in drought indicators for europe and worldwide.

3.7 Land Use Data

  • Copernicus Global Land Service The Copernicus Global Land Service provides information on land cover and land use at a global scale. It is based on satellite data and provides information on land cover, land use, and changes in land cover over time. The data is available as raster files and can be accessed via an API.
  • EuroCrops Dataset from Zenodo is a open source dataset which tries to provide information about the crop type of each individual field in a unified an machine-readable syntax.

3.8 Location / Mobility Data

  • openroute service provides an API and a R package to access routing and isochrone data.
